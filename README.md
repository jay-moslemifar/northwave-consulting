# Northwave Consulting

A bilingual (EN/NL) marketing website for a fictional consulting company, built as a freelance portfolio project.

This project focuses on clean structure, performance, and professional front-end implementation rather than visual experimentation or feature-heavy complexity.

---

## Project Goals

- Demonstrate a real-world freelance-ready setup
- Clean, maintainable SvelteKit project structure
- Bilingual content (English / Dutch)
- Mobile-first, accessible, and performant UI
- Minimal dependencies and deliberate tooling choices

---

## Tech Stack

- **Framework:** SvelteKit
- **Styling:** Tailwind CSS
- **Formatting:** Prettier
- **Linting:** ESLint
- **Deployment target:** Vercel
- **Package manager:** npm

---

## Scope

- Pages: Home, About, Contact
- Bilingual routing (`/en` and `/nl`)
- Static marketing site (no backend, no authentication)

This repository currently contains the initial project scaffolding and configuration only.

---

## Development

Install dependencies:

    npm install

Run the development server:

    npm run dev -- --open

---

## Notes

This is a fictional company created solely for demonstration purposes.
